# Aplikasi Todo List

## Deskripsi Proyek

Aplikasi Todo List adalah aplikasi berbasis web sederhana yang digunakan untuk mencatat, menampilkan, dan mengelola daftar tugas harian. Aplikasi ini dibuat menggunakan **PHP Native**, **MySQL/MariaDB**, serta dijalankan pada server lokal menggunakan **Laragon**.

Proyek ini dibuat untuk memenuhi tugas mata kuliah **Back-End Development** (Semester 3) dan dikerjakan secara berkelompok dengan pembagian tugas yang jelas pada setiap anggota.

---

## Teknologi yang Digunakan

* PHP Native
* MySQL / MariaDB
* HTML & CSS
* JavaScript (sederhana)
* Laragon (Local Server)
* Visual Studio Code
* GitHub

---

## Fitur Aplikasi

Fitur-fitur yang tersedia pada aplikasi ini antara lain:

* Menambahkan tugas baru (Create)
* Menampilkan daftar tugas (Read)
* Menghapus tugas (Delete)
* Menampilkan status tugas

Catatan:

> Fitur edit atau update tugas belum tersedia pada versi aplikasi ini dan dapat dikembangkan pada versi selanjutnya.

---

## Pengujian Aplikasi

Pengujian aplikasi dilakukan menggunakan server lokal Laragon untuk memastikan seluruh fitur berjalan dengan baik tanpa error.

### Hasil Pengujian

| Fitur  | Status         |
| ------ | -------------- |
| Create | Berhasil       |
| Read   | Berhasil       |
| Update | Belum tersedia |
| Delete | Berhasil       |

Berdasarkan hasil pengujian, aplikasi berjalan dengan baik dan tidak ditemukan error pada fitur yang tersedia.

---

## Cara Menjalankan Aplikasi

1. Jalankan Laragon
2. Aktifkan Apache dan MySQL
3. Letakkan folder project di direktori:

   ```
   C:\laragon\www\
   ```
4. Import database ke phpMyAdmin
5. Buka browser dan akses:

   ```
   http://localhost/Aplikasi-To-Do-List-25/public
   
   ```
